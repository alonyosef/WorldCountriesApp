# WorldCountriesApp
With this app you will be able to scroll through all the countries of the world (courtesy of https://restcountries.eu/)

You can:
* Find countries' native names (written in the local dialect)
* Find bordering countries
* Sort your finding according to the name of the country or its area size

Etc...


# System requirements
The app was built using Android Studio 3.4.2 and it targets the latest devices\android OS (pie)


# Key components and usages
* Fragment
* RecyclerView
* Volley


# How to run ?

* Clone the repository to your computer
* The apk file for installation of the app can be found in Countries\app\release\WorldCountries.apk
* You can also open the project via Android Studio and run it using Shift+F10
